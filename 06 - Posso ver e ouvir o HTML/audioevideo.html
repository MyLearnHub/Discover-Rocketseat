Posso ver e ouvir o HTML

<video>

Utilizada para trazer videos ao HTML

Importante o uso do atributo controls, ele traz os controles padrões do navegador para acessibilidade do usuário

Vídeo sem o atributo controls:



<source>

A tag source também indica uma mídia, porém ela explica ao navegador o tipo daquela mídia com o atributo type





Atributos que acompanham a tag <video>

autoplay e muted

Faz o vídeo iniciar automaticamente e mutado respectivamente


heigth e width

Define a altura e largura do vídeo, mas melhor usar o CSS


preload

Define o comportamento e por tabela a qualidade de reprodução do vídeo

  • valor "auto" - Faz o preload do vídeo antes do usuário apertar no botão, adiantando seu desempenho
  • valor "metadata" - Faz o preload apenas do necessário para o vídeo funcionar
  • valor "none" - Não faz preload de nada, espera o play do usuário para processá-lo

loop

Faz o vídeo se comportar em loop, reiniciando automáticamente ao finalizar


poster

Define uma thumbnail para o vídeo




<audio>

Em suma funciona da mesma maneira que a tag <video> mas para audios, inclusive possuindo quase os mesmos atributos



<iframe>

Significa inline frame, é um elemento que traz um conteúdo externo

Além dos atributos padrões de vídeo esse elemento também traz:

  • frameborder - Define se o vídeo tem ou não borda
  • allow - significa permitir, traz algumas definições padrões da origem desse vídeo que é possível configurar
  • allowfullscreen - permite a tela cheia

Importante lembrar que essa tag também funciona para mapas, áudios e qualquer mídia que suporte exportat iframe
