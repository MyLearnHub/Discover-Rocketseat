Guia estelar de CSS

Anatomia do CSS

A anatomia do CSS consiste em 4 partes, são elas:
Seletor - Ex: h1.
Declaração - Ex: color: red.
Propriedade - Ex: font-size:.
Valor da Propriedade - Ex: 60px.

Um exemplo de elemento sendo estilizado: 

            h1 {
                color: blue;
                font-size: 60px;
                background: gray;
            }


Seletores

Os principais tipos de seletores são:

  • Seletor Global: * - Seleciona todos os elementos da página
  • Seletor de Elemento: h1, h2, p, div
  • Seletor de ID: #box, #container
  • Seletor de Class: .red, .m-4

Especificidade

É um cálculo matemático, onde, cada tipo de seletor e origem do estilo, possuem valores a serem considerados.

  • 0 - Seletor Universal, combinadores e pseudo-classes de negação
  • 1 - Seletor de Elemento e pseudo-elementos
  • 10 - Seletor de class e atributos
  • 100 - Seletor de ID
  • 1000 - CSS Inline
  • !important - Absoluto

At-rules

Define e se relaciona com alguns comportamentos do CSS, sempre indicado pelo caractere "@"

  • @import - Incluir um CSS externo
  • @media - Regras condicionais para dispositivos
  • @font-face - Fontes externas
  • @keyframes - Animações

Shorthand

Tem como finalizade resumir códigos, juntar propriedades e tornar o código mais legível.

Propriedades padrão:

background-color: #f1f1f1;
background-image: url(images/exemploimage.jpg);
background-repeat: no-repeat;
background-position: left top;

Shorthand:

background: #f1f1f1 url(images/exemploimage.jpg) no-repeat left top;

Funções

Lidam com valores de maneira mais fácil no CSS

@import url('http://urlaqui.com/style.css')
color:rgb(255, 124, 0)
width: calc(100% - 10px)

Vendor Prefixes (Prefixos de Navegador)

Servem para interagir com elementos específicos em navegadores

  • -webkit - Chrome, Safari, IOS e Android
  • -moz - Mozilla (Firefox)
  • -ms - Internet Explorer
  • -o - Opera
