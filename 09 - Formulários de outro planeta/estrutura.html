Estrutura

<form>

Elemento acompanhado dos atributos padrões action e method


action

Usado para enviar os dados coletados no formulário para algum lugar, se vazio, ele envia para a própria página atual


method

Usado para determinar se é um metódo GET ou POST


<fieldset>

Usado para fazer agrupamento de campos

Acompanhado de 3 principais atributos, disabled, form e name

  • disabled - Desabilita todos os elementos dentro dele, não submete ao formulário
  • form - Qual formulário esse fieldset pertence (indicado por um id no formulário)
  • name - O nome do grupo de fieldset

<legend>

Usado para indicar o nome do agrupamento visualmente, primeiro elemento do fieldset

Dados do usuário

<label>

É usada para referênciar uma input


<button>

Cria um botão dentro do formulário

Atributos importantes:

  • type - define a função daquele botão, button (padrão), reset (reseta os dados nas inputs) e submit (envia os dados do formulário)
  • autofocus - responsável por chamar a atenção do navegador ao entrar na página, facilitando o envio
  • disabled - desabilita o botão
  • name - da nome ao botão
  • value - coloca um valor no botão
  • form - indica o formulário do botão

<datalist>

Semelhante ao select, ele cria uma lista de opções (dentro de uma input), permitindo a digitação para um novo valor
