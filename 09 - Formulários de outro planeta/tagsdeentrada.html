Tags de Entrada

<input>

autocomplete

Usado para trazer informações que usuário usa com frequência no navegador, e autocompletar



readonly

Permite ao usuário apenas ler o conteúdo, mas não editá-lo

Diferença do disabled:



pattern

Permite uma validação na própria input, como um RegEx



inputmode

Altera o tipo de teclado



multiple

Aceita multiplos valores em um campo




size

Modifica o tamanho do campo de caracteres aceitos




spellchecker

Habilita ou desabilita a verificação ortográfica



accept

Determina o tipo de arquivo que a input aceita



checked

Deixa marcado as inputs do tipo checkbox e radio

confirma o check


min / max

Delimita o valor númerico minimo e máximo em uma input



step

Define o intervalo dos botões de ajuda numa input numérica



<textarea>

Semelhante a input, cria uma caixa de texto maior para digitação, seu tamanho de caracteres suportados é definido pelos atributos rows e cols



wrap

Define a quebra de linha da textarea



<optgroup>

Cria um grupo de opções dentro de um select
