Posicionando Foguetes

Page Layouts

  • Tables - Usado antigamente para posicionar elementos
  • Floats e clear - Sistema de elementos flutuante, adaptado, mas em desuso da forma original
  • Frameworks e Grid Systems - Maioria em desuso
  • Flexbox - Muito usado hoje em dia ainda
  • Grid - Modelo mais atualizado de posicionamento de elementos em CSS

Posicionamento

Controla na página o elemento vai se posicionar, alterando o fluxo padrão dos elementos.

values: static (padrão) | relative | absolute | fixed


relative

Entra em conjunto com as propriedades: top, right, bottom, left e z-index



absolute

Ele torna o elemento relativo ao corpo da página, fazendo os outros elementos ignorarem sua posição inicial. Ele pode se tornar relativo a um elemento pai, se este tiver um position: relative



Grid

Posiciona os elementos dentro da caixa, possibilitando o posicionamento horizontal e vertical ao mesmo tempo. Pode ser flexível ou fixo. Cria espaço para os elementos filhos habitarem.
