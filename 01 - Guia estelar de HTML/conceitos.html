Guia estelar de HTML

O que é HTML?

HTML significa Hypertext Markup Language, é uma linguagem de marcação, semântica. Amplamente usada para estruturar sites Web.


Tags

Exemplo de tag: <h1>"Título aqui"</h1>

Cada uma das tags são chamadas também de elementos, complementar a elas existem os atributos


Um exemplo de elemento com atributos: <img src="" alt="">

Nesse exemplo acima o elemento é a tag "img" e os atributos são "src" e "alt"


Atributos

Vale a pena mencionar os atributos booleanos, eles não necessitam de argumentos (ou conteúdo)

Exemplo: atributo "disable" usado em uma input para desabilitar



Atributos globais

Os atributos globais, são aqueles que podem ser utilizados por todas as tags HTML, existem vários deles.

Veja a seguir uma lista dos atributos globais mais utilizados:

  • class
  • contenteditable
  • data-*
  • hidden
  • id
  • style
  • tabindex
  • title

contenteditable
