Terminologia

Flex Container

É flexível, ou seja, contém algo dentro dele


Flex Item

Determinao item (conteúdo) dentro do flex container


Nesting

Termo para determinar um elemento que contem outros elementos


Axis

Indica o eixo

Main

Eixo padrão na horizontal, possui início e fim

Cross

Eixo na vertical, possui início e fim


Flex-Sizing

Define um tamanho flexível, altera a largura e altura dos items para preencher o container
