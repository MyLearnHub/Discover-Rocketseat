block x inline

Define como os elementos se comportam em relação a outros elementos

display: block

  • Ocupa toda a linha colocando o próximo elemento pra baixo
  • width e heigth são respeitados
  • padding, margin e border tem funcionamento padrão

display: inline

  • Elemento um ao lado do outro
  • width e heigth não funcionam
  • somente valores na horizontal de padding, margin e border
